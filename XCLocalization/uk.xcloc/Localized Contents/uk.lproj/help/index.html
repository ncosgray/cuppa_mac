Cuppa

Cuppa - це невеликий застосунок для macOS, що дозволяє відстежувати час приготування чашки чаю. Набридло залишати чай занадто довго, щоб він став гірким і холодним, або пити його занадто рано і не оцінити весь його потенціал? Тоді ця утиліта для вас!

Особливості

  • Весь функціонал доступний з головного меню.
  • Піктограма док-станції може відображати невеликий таймер зворотного відліку, коли напій заварюється.
  • Налаштуйте список напоїв та час заварювання.
  • Функція швидкого таймера перетворює Cuppa на універсальний таймер довільної форми.
  • Налаштуйте сповіщення про повне заварювання.
  • Також чудово підходить для приготування кави у френч-пресі та локшини швидкого приготування!
  • Доступний повний вихідний код Cocoa.
  • Працює на macOS 10.9-11.0.

Визначення часу заварювання

Використовуйте його з док-панелі: просто клацніть правою кнопкою миші або клацніть контрольною кнопкою миші на піктограмі Cuppa і виберіть напій, який ви готуєте. Або виберіть напій з меню Напої. Cuppa почне відлік часу заварювання, і ви побачите, як у чашці з'явиться пакетик чаю, який поступово темнішатиме в міру того, як чай настоюватиметься. Коли чай буде готовий, Cuppa спробує привернути вашу увагу різними способами, які можна налаштувати. Ви також можете увімкнути таймер зворотного відліку, який відображається у вигляді маленького значка на іконці Док-станції.

Налаштування Cuppa

Чай - річ особиста, але Cuppa можна налаштувати на свій смак. Для цього виберіть у меню Налаштування....

Список напоїв: Вкладка Список напоїв дозволяє змінити будь-який наявний чай, двічі клацнувши по його назві або часу заварювання і просто ввівши нове значення. Ви також можете повністю видалити напої або додати нові, натиснувши кнопки під списком напоїв.

При натисканні на вкладку Опції відкривається сторінка додаткових налаштувань конфігурації:

  • Bounce Dock icon: When brewing is complete, bounce the Dock icon repeatedly until you click it. This is annoying enough to get your attention.
  • Play sound: If you prefer an audio notification, Cuppa will make a clinking-spoon sound when the timer hits zero. You'll also hear quiet beeps for the last few seconds of steep time.
  • Show "Brewing Complete" alert box: Show an alert box on the screen when your beverage is done. This alert box will also allow you to quit Cuppa with a single click (or keystroke) when your tea is done.
  • Notify via Notification Center: On macOS 10.8 or later, use the built-in Notification Center to show "Brewing Complete" messages. If you find you're missing notifications because they disappear too quickly, try setting the Cuppa alert style to "Alerts" in Notifications System Preferences.
  • Show countdown timer on Dock icon: When this option is enabled, Cuppa will show a tiny countdown timer as part of its Dock icon. Try not to strain your eyes looking at it!
  • Include steep times in menus: Check this box if you want Cuppa to include times in the beverage menu items, so that before the timer even starts, you know how long you'll have to wait.
  • Check for program updates automatically: If this is enabled, Cuppa will periodically contact a web service and inform you when a new version of the program is available to download.

Cuppa Quick Timer

What if you need to brew a cup of tea in a hurry, and don't have time to add a new beverage in Preferences? No worries; Cuppa's Quick Timer feature is a simple way to start a timer in a jiffy. Just select Quick Timer... from the menu and type either a number of seconds (e.g. "30"), minutes and seconds (e.g. "2:45"), or any time up to the maximum 10 hours. Cuppa does the rest, and notifies you when time is up.

Miscellaneous

To cancel the timer while it is running, select Cancel from the menu.

Customizing Sounds

You may find it amusing if your computer announces, "Tea -- Earl Grey -- Hot" when your morning tea is ready to drink. Although there is no mechanism within Cuppa's Preferences for changing the alert sounds, it's actually quite easy to accomplish this. You'll need a couple of AIFF format sound files to copy into the Cuppa application bundle. These will replace the default sounds for beginning and completing brewing. Name them pour.aiff and spoon.aiff, respectively.

Then, replace the default sound files as follows:

  1. Quit Cuppa if it is running.
  2. Find the Cuppa application in your Applications folder and right-click (control-click) it.
  3. Click Show Package Contents.
  4. Navigate to Contents and then to Resources.
  5. Make backup copies of the pour.aiff and spoon.aiff sound files, just in case something horrible happens.
  6. Copy your pour.aiff and spoon.aiff sound files into the Resources folder.
  7. That's it! Now you can run Cuppa and test your modifications.
